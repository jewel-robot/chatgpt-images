git切换远程分支

You
我git clone了这个：https://github.com/ROBOTIS-GIT/turtlebot3.git，我想把它切换到humble版本
AI

可以的。ROBOTIS-GIT/turtlebot3 这个仓库本身就有一个 humble 分支（还有 jazzy、main(rolling)）。 GitHub

你在本地把它切到 Humble 分支，按下面做就行：

cd turtlebot3

# 1) 拉取远端最新分支/标签信息
git fetch --all --tags

# 2) 看看远端有哪些分支（确认有 origin/humble）
git branch -r | grep humble

# 3) 切换到 humble（推荐用 git switch）
git switch humble 2>/dev/null || git switch -c humble --track origin/humble

# 4) 验证当前分支
git branch --show-current
git status

如果你看到当前分支是 humble，就切换成功了。

补充一点：TurtleBot3 通常还会配套用到 turtlebot3_msgs、turtlebot3_simulations 等仓库，它们也同样有 humble 分支。 GitHub+1
