卡尔曼滤波相关知识并不需要太多介绍，那么如何实现编程卡尔曼滤波，有太多的方法，网络上也有是各种资源介绍，在网络上检索了一些，进行整理，重点阐述的是一些常见的库文件或者具体的代码实现，比如MATLAB中的kalman函数就不介绍。资源整理大体如下，

FilterPy

可以说是Python中最为流行、最为完善的卡尔曼滤波库之一。支持的滤波器有KF、EKF、UKF等。

特点：易于使用，文档详细，示例丰富，提供了各种用于调试和分析的工具。

https://filterpy.readthedocs.io/en/latest/filterpy.readthedocs.io/en/latest/

PyKalman

又是一个Python卡尔曼滤波库，提供了一些基础的卡尔曼滤波实现。

特点：相对简单，易于理解，但功能不如FilterPy丰富。

https://github.com/pykalman/pykalmangithub.com/pykalman/pykalman

TinyEKF

C/C++开发，兼容嵌入式平台。

特点：轻量级设计，静态内存分配，无动态内存需求，适合Arduino、STM32等资源受限设备。提供Python接口用于原型验证，支持快速算法迭代。示例应用包括无人机姿态控制和传感器数据融合。

KF-GINS

语言：C

特点：专注于GNSS/INS（全球导航卫星系统/惯性导航系统）组合导航，采用21维误差状态模型。包含IMU误差补偿、惯性导航解算和卡尔曼滤波全流程，支持高精度定位。

Kalman Filter Library

特点：C++开发，基于Eigen支持多种滤波器类型，用户可自定义系统状态、控制向量、系统模型和测量模型，支持线性和非线性系统

BFL (Bayesian Filtering Library)

特点：一个通用的贝叶斯滤波库，支持卡尔曼滤波、粒子滤波等。模块化设计，适合机器人和传感器融合。

KalmanJS

特点：轻量级的JavaScript卡尔曼滤波库，适合Web应用，简单易用，适合实时数据平滑。

Kalman filter

特点：包括连续-离散扩展卡尔曼滤波器，并鼓励贡献更多的滤波器。它提供了 MATLAB、C++ 和 Python 的实现。
