怎么让这个autohotkey脚本开机自启动

要让这个 AutoHotkey 脚本（autohotkey.ahk）开机自启动，你可以根据需要选择以下三种方法之一（Windows 下适用）：

✅ 方法一：放入「启动」文件夹（最简单）

  1. 按 Win + R 打开「运行」。

  2. 输入以下命令并回车：

    makefile
    shell:startup

    这会打开当前用户的启动文件夹。

  3. 把你的脚本快捷方式（不是脚本本身）放进去：

    • 右键 autohotkey.ahk → 创建快捷方式；

    • 然后把这个快捷方式复制到刚刚打开的 启动文件夹 中。

  4. 以后每次开机登录时，Windows 就会自动运行这个脚本。

✅ 方法二：用 AutoHotkey 编译成可执行文件（更稳定）

  1. 右键 autohotkey.ahk 文件 → 选择 “Compile Script”（编译脚本）。
    这会生成一个 autohotkey.exe 文件。

  2. 然后把这个 .exe 文件的快捷方式放到上面的启动文件夹中（shell:startup）。

这种方式的优点是：

  • 不需要系统安装 AutoHotkey；

  • 启动速度稍快。

✅ 方法三：通过注册表添加启动项（适合高级用户）

  1. 按 Win + R → 输入：

    nginx
    regedit

  2. 打开注册表路径：

    mathematica
    HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run

  3. 右键右侧空白处 → 新建 → 字符串值，命名为例如 AutoHotkey。

  4. 双击它，在“数值数据”中填写脚本的完整路径，比如：

    arduino
    "C:\Users\<你的用户名>\Documents\AutoHotkey\autohotkey.ahk"

⚙️ 可选：让脚本以管理员权限运行

如果脚本里需要控制系统功能（如模拟 Ctrl+Alt+Del 或控制系统设置），则建议：

  1. 右键快捷方式 → 属性 → 高级；

  2. 勾选 “以管理员身份运行”。
