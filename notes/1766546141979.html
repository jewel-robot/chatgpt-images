另外，官方博客也写过：XR exporter 在 2024 SP0 又“回到”SOLIDWORKS 和 Visualize 里——这也解释了你在 2023 里找不到的落差。 The SOLIDWORKS Blog

下面给你两条“在 2023 版本可落地”的做法（按推荐程度排序），每一步写清楚用什么软件、怎么点。

方案 A（最推荐）：用 SOLIDWORKS Visualize 导出 .glb/.gltf

这是目前最稳、最少踩坑的路线。Visualize 官方就支持 File > Export > Export Project > Extended Reality（GLTF/GLB）。 The SOLIDWORKS Blog+1

A1. 你需要的软件

  • SOLIDWORKS Visualize（Standard 或 Professional 都行）

    • 导出 GLB/GLTF：Standard/Pro 都能做；动画等高级能力主要在 Pro。 The SOLIDWORKS Blog

  • SOLIDWORKS 里通常也会有 SOLIDWORKS Visualize Add-in（把几何/外观/贴花等送到 Visualize）。 SOLIDWORKS Web Help

A2. 在 SOLIDWORKS 2023 里操作

  1. 打开你的零件/装配

  2. Tools > Add-Ins…

  3. 勾选 SOLIDWORKS Visualize（如果列表里有）

    • 这个 Add-in 的作用是把几何/外观/贴花更顺畅地传到 Visualize。 SOLIDWORKS Web Help

  4. 用 Visualize Add-in 把模型发送到 Visualize（一般在界面里会出现 Visualize 相关按钮/菜单）

如果你 Add-Ins 里也没有 Visualize：那通常是 Visualize 没安装（不是没许可证）。需要用安装管理器把 Visualize 组件装上。

A3. 在 SOLIDWORKS Visualize 里导出 GLB/GLTF

  1. 打开 Visualize，确认模型已经导入/同步进来

  2. File > Export > Export Project

  3. 在导出格式里选：Extended Reality（GLTF/GLB）

  4. 选择导出为 .glb（一个文件把贴图打包进去更方便）或 .gltf（会生成外部纹理文件夹）

方案 B：安装“SOLIDWORKS XR Exporter”独立插件（有时可用，但不如 A 稳）

有经销商/教程提供过“单独下载安装 XR Exporter，然后在 Tools > Add-Ins 勾选”的流程。 GSC+1
提醒你：这条路在不同年份版本/安装环境里成功率不如 Visualize 稳，而且你现在已经验证 2023 里没出现，所以我把它作为备选。

大致步骤：

  1. 从你们的订阅下载/经销商下载页面拿到 XR Exporter 安装包（有些文章说在下载列表里能找到） GSC

  2. 安装完成后重启 SOLIDWORKS

  3. Tools > Add-Ins… 里勾选 SOLIDWORKS XR Exporter GSC

  4. 然后 File > Save As 里会出现 Extended Reality (*.gltf) / Extended Reality Binary (*.glb)（如果插件正确装上）

你现在该怎么选

  • 你要“今天就能导出 GLB/GLTF，并尽量保留外观/贴花/纹理”：走 **方案 A（Visualize）**最省事最稳。 The SOLIDWORKS Blog+1

  • 你们之后会升级到 2024：那 XR exporter 在 2024 SP0 起官方说已经“在 SOLIDWORKS 里也能用”。 The SOLIDWORKS Blog
