LTctfer #1

1.首先是反代项目选择，我个人用的是2api，也就是下面这个项目

github.com

GitHub - justlovemaki/AIClient-2-API: Simulates Gemini CLI, Antigravity, Qwen Code, and...

Simulates Gemini CLI, Antigravity, Qwen Code, and Kiro client requests, compatible with the OpenAI API. It supports thousands of Gemini model requests per day and offers free use of the built-in Claude model in Kiro. Easily connect to any client via the API, making AI development more efficient!

项目部署比较简单，docker一键部署即可，部署在自己本机或者虚拟机上都可以，贴个docker命令给大家

docker run -d -p 3000:3000 -p 8085:8085 -p 8086:8086 -p 19876-19880:19876-19880 --restart=always -v “./config:/app/configs” --name aiclient2api justlikemaki/aiclient-2-api

部署完成后需要去web端完成下一步，地址：http://localhost:3000/

2.然后是如何进行反代

进入web端后进入下面的界面，选择模型提供商（反代kiro的话就Claude Kiro OAuth）,其他的如果是自用就不用修改

接着进入提供商池管理，在这里进行kiro的注册和授权


点击这个生成授权

会跳转到这个

点一下会生成一个授权界面

然后就直接在浏览器打开注册亚马逊即可

注册后会生成一个kiro-auth文件，自动关联即可

3.接下来是接到cc中

在配置管理界面中有claude code的协议（也就是api地址），也就是：http://localhost:3000/claude-kiro-oauth 然后填入apikey即可（同样在配置管理界面修改）

接下来就可用啦

4.下面是如何获得无限qq邮箱的教学（实测下来qq的英文邮箱基本不会被封的）

首先点进qq邮箱，点击右上角的头像，然后点击账号和安全

里面会有一个qq英文邮箱
用这个邮箱去注册kiro，注册后删除再创建即可
哦在亚马逊授权的时候记得先把上一个账号退出一下
好啦，接下来就可以畅快使用啦，实测下来速度还行，代码质量也还可以，各位如果公益站额度用光了可以用这个
